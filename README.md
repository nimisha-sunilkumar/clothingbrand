**Style Aura â€“ Trendy Fashion** is a **responsive online clothing store** built with **HTML, CSS, and JavaScript**.

## Features
- **Responsive Design:** Works seamlessly on desktops, tablets, and mobile devices.  
- **Product Catalog:** Browse clothing items by category with images and descriptions.  
- **Shopping Cart:** Add or remove items and view total price dynamically.  
- **Price Calculation:** Total price updates automatically based on selected items.  
- **User Details Form:** Collect customer information including name, address, phone number, and email.  
- **Payment Options:** supports Paytm, PhonePe, Google Pay, and Cash on Delivery (COD).  
- **Order Confirmation:** Users receive confirmation after placing an order.
- **Category Filtering:** Easily filter products by type or category.
- **Cart Icon Badge:** Shows number of items in the cart in real-time.  
- **Modern UI/UX:** Clean layout with attention to user interaction and aesthetics.
